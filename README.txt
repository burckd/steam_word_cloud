Steam Playtime Word Cloud

This Python script generates a word cloud based on your Steam game playtimes. The more you have played a game, the bigger its name appears in the word cloud.


Installation & Setup

1. Install Dependencies

pip install requests wordcloud matplotlib numpy

2. Set API Key & Steam ID in Script

Open steam_word_cloud.py and replace the placeholders with your actual API key and Steam ID:

STEAM_API_KEY = "INSERT HERE"
STEAM_ID = "INSERT HERE"

4. Run the Script

python steam_word_cloud.py
